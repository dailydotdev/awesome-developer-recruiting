# Promotion Playbook

Use this document to execute the Day 1-7 promotion tasks from `PLAN.md` consistently.

## Core Positioning

- Focus: developer and engineer hiring only (not generic recruitment).
- Value: practical, curated tooling and resources across sourcing, assessment, interviewing, and hiring operations.
- Angle: open-source and community-driven list under the `dailydotdev` organization.

- Repository: `https://github.com/dailydotdev/awesome-developer-recruiting`
