## Sourcing Platforms

*Platforms for finding and reaching out to developer candidates.*

- [LinkedIn Recruiter](https://www.linkedin.com/talent/hire) - Professional network with advanced search filters for finding developers by skills, company, and experience.
- [daily.dev Recruiter](https://recruiter.daily.dev) - Developer hiring platform built on the trust of 1M+ active developers, using behavioral signals and opt-in introductions.
- [SeekOut](https://seekout.com) - AI-powered talent search with deep technical profile aggregation from GitHub, Stack Overflow, and patents.
- [hireEZ](https://hireez.com) - AI sourcing platform with Boolean search across 45+ open web platforms and 800M+ candidate profiles.
- [Hired](https://hired.com) - Marketplace where developers set salary expectations and companies apply to them with upfront offers.
- [Toptal](https://www.toptal.com) - Freelance network of pre-screened top 3% developers available for contract and full-time roles.
- [Turing](https://www.turing.com) - AI-matched remote developer sourcing from a global pool of vetted engineers.
- [Wellfound](https://wellfound.com) - Startup-focused talent platform where developers browse startup jobs with salary and equity transparency.
- [Stack Overflow Talent](https://stackoverflow.co/talent) - Developer recruitment advertising and employer branding on the largest developer Q&A community.
- [Gun.io](https://gun.io) - Vetted freelance developer marketplace with a focus on senior engineers and quality matching.
- [Arc.dev](https://arc.dev) - Remote developer hiring platform with pre-vetted candidates and HackerRank assessments built in.
- [Dice](https://www.dice.com) - Technology-focused career platform with employer branding and candidate matching for tech roles.

## Technical Assessment

*Platforms for evaluating developer coding skills and technical ability.*

- [HackerRank](https://www.hackerrank.com) - Coding challenges and technical assessments covering 35+ programming languages with plagiarism detection.
- [Codility](https://www.codility.com) - Automated coding tests with real-world task simulations and anti-cheating measures.
- [CodeSignal](https://codesignal.com) - Skills evaluation platform using coding scores and framework-specific assessments validated by research.
- [LeetCode](https://leetcode.com) - Developer skill practice platform widely used to benchmark algorithm and data structure knowledge.
- [CoderPad](https://coderpad.io) - Live collaborative coding environment for technical interviews with 30+ language support.
- [TestGorilla](https://www.testgorilla.com) - Pre-employment testing platform with coding tests, cognitive ability, and culture-add assessments.
- [Qualified](https://www.qualified.io) - Technical assessment platform featuring project-based challenges that mirror real development work.
- [Coderbyte](https://coderbyte.com) - Coding challenge platform with 300+ challenges and AI-powered code scoring for recruiters.
- [Devskiller](https://devskiller.com) - RealLifeTesting methodology using project-based assessments that simulate actual development tasks.
- [Exercism](https://exercism.org) - Free, open-source coding practice platform with mentor-reviewed exercises in 70+ languages.
- [Skills Checklist Builder by daily.dev](https://recruiter.daily.dev/toolkit/skills-checklist) - Free interactive skills checklists for evaluating developer candidates by role and stack.
